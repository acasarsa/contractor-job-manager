## Building an app for tracking small team contractor / renovation projects. 

- Timesheet tracker that can allow for task input across multiple job sites and multiple tasks.

- Can track job status / costs etc over life of job.
- Can utilize AI parser to import / parse materials / subcontractor costs into tracker from pdf or photos
- keeps track of bid numbers and compares to actual costs.
- creates Client version of Bid vs Actuals updates to send to client
- auto-calculates and updates numbers from the Timesheet and imported materials and subcontractors
